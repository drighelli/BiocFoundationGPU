## BiocFoundationGPU

This package allows deploying a set of pre-defined foundation models for various tasks in R, 
leveraging GPU acceleration for improved performance. It provides an interface to easily access 
and utilize these models for tasks such as generating embeddings of spatial or imaging datasets. 

The package is designed to be user-friendly and efficient, making it accessible to both beginners 
and experienced users in the field of machine learning and deep learning.
